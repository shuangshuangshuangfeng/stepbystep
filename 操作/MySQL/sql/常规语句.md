# 常用语句

（1）查找重复数据有多少

```sql
select count(*) from lj_ticket group by lj_ticket_id having count(id) > 1
```

（2）查询结果进行拼接

```sql
select group_contact(id) from lj_ticket where id_deleted = 0
```



（3）根据条件删除

```sql
delete from lj_ticket 
where XXX
```

（4）根据条件更新

```sql
update lj_ticket
set is_deleted = 0
where XXX
```
